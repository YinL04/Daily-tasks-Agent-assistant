input: AgentContext
output: UrlReference[]
---

## 目标

整理执行计划时需要访问或参考的网址，让用户后续行动更顺手。

## 规则

- 从用户输入中提取 `http` 和 `https` 链接。
- 尝试根据用户描述的具体任务场景推荐有帮助的参考链接。
- 推荐的链接必须与用户输入直接相关，不要推荐无关的通用网站。
- 说明推荐理由时要结合用户的具体场景。
- 当前实现不会联网核验推荐链接是否实时可用；涉及关键事务时应让用户二次确认。

## 输出要求

返回 `UrlReference[]`，每项包含标题、URL、推荐原因和分类。
